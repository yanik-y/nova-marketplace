Drop product photos here, named by the item ID from data/items.csv.

Examples:
  1.jpg   matches item with id=1 (Winter jacket)
  2.png   matches item with id=2 (Desk lamp)
  3.jpeg  matches item with id=3 (Plant pot)
  14.webp matches item with id=14

Supported extensions: jpg, jpeg, png, webp.

If a photo for the item ID is found here, it replaces the placeholder
from uploaded_images/. If you do not drop a file in here for that ID,
the original placeholder is shown.

To find the right ID for an item, open data/items.csv and look at the
first column.
